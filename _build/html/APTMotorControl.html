GetStageAxisInfo_MaxPos(channel=0)[source]¶

Get the maximum position of the stage that is accessible using the MoveAbsoluteEx or MoveRelativeEx commands, although you may be able to exceed it by Jogging. I think this is a user-settable quantity. For the small stepper we have, the max travel is like 18mm. (Or should be ~25mm?)

GetStageAxisInfo_MinPos(channel=0)[source]¶

Get the minimum position of the stage that is accessible using the MoveAbsoluteEx or MoveRelativeEx commands, although you may be able to exceed it by Jogging. I think this is a user-settable quantity. For the small stepper we have, if it’s been “homed” then it sets 0 to be the minimum position.
