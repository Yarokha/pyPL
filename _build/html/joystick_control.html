Joystick Control¶

joystick_control.StartControl(app, positioners)[source]¶

Initialize pygame and start the monitor_joystick event loop.

joystick_control.calc_stepsize(Z, motors_or_piezos, fine_control)[source]¶

throttle position sets the step size... algorithm will likely need fine tuning

joystick_control.center_piezos(positioners)[source]¶

Center the piezos within their range.

joystick_control.get_joystick_position(joystick)[source]¶

Returns joystick position in X,Y, and Z (where Z is throttle). All three of these values should be in the range -1 to 1

joystick_control.handle_joystick_button_down(event, positioners, wobbler, autoit)[source]¶

Deals with a joystick button press.

joystick_control.handle_joystick_button_up(event)[source]¶

Deals with release of a joystick button (although there may not be anything meaningful that happens as a result of this.

joystick_control.init_autoit()[source]¶

Initialize AutoIT module.

joystick_control.monitor_joystick(app, positioners, joysticks)[source]¶

Monitor the joystick for changes in position or button presses.

joystick_control.toggle_fine_coarse_control()[source]¶

Toggle between fine and coarse control (changes how the step size is calculated in calc_stepsize).

joystick_control.toggle_motor_piezo_control()[source]¶

Toggle between controlling the motors or the piezos.

joystick_control.update_position(positioners, xyz)[source]¶

The joystick has been moved, so update the appropriate motor or piezo position accordingly and send notice of the event to the main pyPL thread.
