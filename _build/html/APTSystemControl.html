APT System Control¶

In an ideal world, this will be a complete working library for the APTMotor

class APTSystemControl.APTSystem(parent, id=-1, pos=wx.Point(-1, -1), size=wx.Size(-1, -1), style=0, name='APTSystem')[source]¶

The System class derives from wx.lib.activex.ActiveXCtrl, which is where all the heavy lifting with COM gets done.

GetHWUnits()[source]¶

runs the activeX command GetNumHWUnitsEx to return the number of hardware units connected.

GetMotorSerialNumbers()[source]¶

runs the activeX command GetHWSerialNumEx to return a list of serial numbers for all of the motor controller cards.

GetNumMotorControllerCards()[source]¶

runs the activeX command GetNumHWUnitsEx to return the number of stepper motor controller cards in the system.

GetNumPiezoControllerCards()[source]¶

runs the activeX command GetNumHWUnitsEx to return the number of piezo controller cards in the system.

GetPiezoSerialNumbers()[source]¶

runs the activeX command GetHWSerialNumEx to return a list of serial numbers for all of the piezo controller cards.
